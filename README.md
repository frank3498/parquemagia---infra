ParqueMagia — Infraestructura DevOps (Terraform + Ansible + Vault)

Este repositorio contiene toda la infraestructura como código (IaC) utilizada para desplegar el proyecto ParqueMagia, un sitio web para parques de diversiones, diseñado bajo prácticas modernas de DevOps, usando Terraform, Ansible y la integración conceptual de HashiCorp Vault para la gestión segura de secretos.

El objetivo principal del proyecto es aprovisionar, configurar y asegurar la infraestructura necesaria en AWS para ejecutar la aplicación, automatizando completamente el proceso.
